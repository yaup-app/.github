# yaup - yet another university planner

## Wichtige Hinweise für die Entwicklung

#### [Conventional Commits](https://www.conventionalcommits.org/de/v1.0.0/)

Den aktuellen Stand könnt ihr immer hier sehen:
https://app.uni-verse.cc

Arbeitet standardmäßig immer auf dem "development"-Branch wenn ihr Änderungen vornehmt, damit wir immer einen stabilen und funktionsfähigen Stand auf dem "main"-Branch haben. Wenn eure Änderungen dann auf dem development-Branch gut genug funktionieren und ihr keine Bugs gefunden habt, dann könnt ihr den development-Branch auch in den main-Branch mergen, um die nächste stabile Version auf dem main-Branch zu haben.

"development"-Branch -> Bei Push werden Änderungen auf https://dev.app.uni-verse.cc sichtbar. Die API ist dann unter https://dev.api.uni-verse.cc erreichbar, die API-Docs unter https://docs.dev.api.uni-verse.cc
"main"-Branch -> Bei Push werden Änderungen auf https://app.uni-verse.cc sichtbar. Die API ist unter https://api.uni-verse.cc erreichbar, die API-Docs unter https://docs.api.uni-verse.cc
